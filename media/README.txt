==============================================
Django Basic Photos
http://code.google.com/p/django-basic-photos/
==============================================

A simple people application for Django projects.

To install this application place somewhere directly
in your Python path and name folder 'photos'. Then add 
'photos' to INSTALLED_APPS in your projects settings.py file.
